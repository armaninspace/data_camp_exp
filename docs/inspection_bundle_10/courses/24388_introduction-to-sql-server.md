# Introduction to SQL Server (`24388`)

## Visible Curated Q/A Pairs

Q: What are counting things?  
A: Counting things means using aggregate functions such as COUNT in SQL Server to count the number of records or items in a dataset.

Q: What is groups, strings,?  
A: Groups and strings refer to using SQL Server aggregate functions (like SUM, COUNT, MIN, MAX, AVG) to group data with GROUP BY, and manipulating text fields (strings) in queries.

Q: What are joining tables?  
A: Joining tables means combining data from two or more tables using INNER, LEFT, or RIGHT joins to retrieve related information from multiple sources.

Q: Why isn't selection box enough on its own?  
A: Selection box, or just using SELECT statements, is not enough on its own because you also need to know how to filter, sort, group, and manipulate data to perform more complex and meaningful data analysis.

Q: What is selection box?  
A: Selection box refers to learning the basics of SELECT statements in SQL Server, which are used to retrieve data from one or more columns, apply filters, and sort results.

Q: How would I transfer selection box to a new but related situation?  
A: You can transfer selection box skills to a new situation by using SELECT statements to retrieve and filter data from different tables or databases, adapting your queries to new data structures or requirements.

Q: Why isn't you've got the power enough on its own?  
A: You've got the power, or just knowing how to create, insert, update, and delete data, isn't enough on its own because you also need to know how to retrieve, filter, and analyze data to fully work with SQL Server.

## Scraped Course Description

### Summary

Learn to use SQL Server to perform common data manipulation tasks and master
common data manipulation tasks using this database system.

### Overview

SELECTion Box

Hit the ground running by learning the basics of SELECT statements to
retrieve data from one or more columns.

You'll also learn how to apply filters to both numeric and text data, and
sort the results.

Groups, strings, and counting things

Herein, you'll learn how to use important SQL Server aggregate functions
such as SUM, COUNT, MIN, MAX, and AVG. Following that, you'll learn how to
manipulate text fields. To round out the chapter, you'll power up your
queries using GROUP BY and HAVING, which will enable you to perform more
meaningful aggregations.

Joining tables

This chapter covers the basics of joining tables, using INNER, LEFT, and
RIGHT joins, so that you can confidently retrieve data from multiple
sources.

You've got the power

In this final chapter, you'll get really hands on! You've worked with
existing tables, but in this chapter, you'll get to CREATE and INSERT data
into them. You'll also UPDATE existing records and practice DELETE
statements in a safe environment. This chapter ensures the course gives you
a thorough introduction to the key aspects of working with data in SQL
Server.
