## Visible Curated Q/A Pairs

Q: Why isn't distributions enough on its own?  
A: Distributions alone are not enough because they only show how one variable is spread out; to fully understand data, you also need to explore relationships between variables and use tools like PMFs and CDFs to get deeper insights.

Q: What should I compare when I use multivariate thinking?  
A: When using multivariate thinking, you should compare how multiple variables interact with each other, such as using multiple regression to describe non-linear relationships or logistic regression to explain and predict binary variables.

Q: What could I misunderstand about multivariate thinking if I only memorize the term?  
A: If you only memorize the term, you might misunderstand that multivariate thinking is just about knowing the definition, rather than actually analyzing and interpreting how several variables interact using methods like multiple and logistic regression.

Q: What is read, clean,?  
A: Read, clean refers to the initial step in a data project where you read the data, check for errors and special cases, and prepare the data for analysis.

Q: Why isn't relationships enough on its own?  
A: Relationships alone are not enough because they only look at two variables at a time; to understand more complex patterns, you need to analyze multiple variables together using multivariate methods.

## Scraped Course Description

### Summary

Learn how to explore, visualize, and extract insights from data.

### Overview

Read, clean, and validate

-The first step of almost any data project is to read the data, check for
errors and special cases, and prepare data for analysis. This is exactly
what you'll do in this chapter, while working with a dataset obtained from
the National Survey of Family Growth.

Distributions

-In the first chapter, having cleaned and validated your data, you began
exploring it by using histograms to visualize distributions. In this
chapter, you'll learn how to represent distributions using Probability Mass
Functions (PMFs) and Cumulative Distribution Functions (CDFs). You'll learn
when to use each of them, and why, while working with a new dataset obtained
from the General Social Survey.

Relationships

-Up until this point, you've only looked at one variable at a time. In this
chapter, you'll explore relationships between variables two at a time, using
scatter plots and other visualizations to extract insights from a new
dataset obtained from the Behavioral Risk Factor Surveillance Survey
(BRFSS). You'll also learn how to quantify those relationships using
correlation and simple regression.

Multivariate Thinking

-Explore multivariate relationships using multiple regression to describe
non-linear relationships and logistic regression to explain and predict
binary variables.
