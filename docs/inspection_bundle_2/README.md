# Inspection Bundle 2

This folder contains sample course review documents generated by the V2
question-generation pipeline.

Included samples:

- [24491_forecasting-in-r.md](/code/docs/inspection_bundle_2/24491_forecasting-in-r.md)
- [24593_visualizing-time-series-data-in-r.md](/code/docs/inspection_bundle_2/24593_visualizing-time-series-data-in-r.md)
- [24594_case-study-analyzing-city-time-series-data-in-r.md](/code/docs/inspection_bundle_2/24594_case-study-analyzing-city-time-series-data-in-r.md)

Why these courses were chosen:

- they are all R time-series courses
- they were the bounded target set for the V2 implementation run
- they provide a useful mix of forecasting, visualization, and case-study
  content
- they are small enough to inspect manually, but different enough to reveal
  whether the generated questions are learner-centered rather than just
  syllabus-shaped

Each file includes:

- generated Q/A pairs
- scraped course summary
- scraped overview
- scraped syllabus text
